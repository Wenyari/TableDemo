# Walkthrough: 前端面试题 (Figma 还原)

## Step 5: Figma 设计稿像素级还原 (100% 对齐)

这是本次开发的最后一步，重点在于完全吃透 Figma 的属性面板，将其转化为精确的 CSS 代码。

### 1. 字体、颜色与边框精准还原
- 我们在 `body` 级别应用了 `"PingFang SC"` 字体，以及 `-webkit-font-smoothing: antialiased;` 保证文字渲染效果。
- 表头文字尺寸精确设置为 `14px` (`#969696`)，而 "整数00x" 则独立设定为 `12px`。
- 我们利用 `::before` 和 `::after` 伪元素在 `table-container` 内部完美还原了纵向交叉的 `0.5px solid #434343` 分割线，而不需要在每个独立的 `div` 上去计算。

### 2. 交互状态的高亮反馈
根据需求，我们深度定制了鼠标的交互反馈：
- **表头按钮**：增加了一层 `padding` 并赋予 `.th-content` 类。在 `:hover` 和 `.active` 时，底色会泛出 `rgba(255, 255, 255, 0.1)` 的毛玻璃微光，同时内部的 `icon-funnel` 图标透明度会从 `0.6` 提升到 `1.0`。
- **Tabs 导航**：非激活态（"订单列表"）设定了特定的毛玻璃背景 `backdrop-filter: blur(27.18px)` 和灰色字体；激活态（"订单统计"）去除了毛玻璃，字体变为全白，并在底部利用绝对定位增加了 `2px` 宽 `70px` 的白色指示条。
- **FilterDropdown 复选框**：我们用纯 CSS 和内联 SVG 结合画出了 14x14 的原生级复选框体验，未选中时只有边框，选中时背景变为白色，SVG 的对勾线条变为深色 `#101010`。

### 3. 毛玻璃特效与层级 (Z-Index)
为了体现设计稿中深邃的科技感，我们广泛运用了 `backdrop-filter: blur(24px)`。
- **侧边栏**：盖在底层。
- **下拉菜单**：`z-index: 10`，悬浮在所有组件之上，配上了大范围的 `box-shadow` 阴影，让浮层更具立体感。

### 结语
项目已完成 100% 的设计稿还原和完整的状态管理逻辑。
