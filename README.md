# 前端工程师笔试题CSS

还原Figma设计稿，表格

## 实现总结

本项目基于 Vite + React 采用模块化组件思想架构（拆分了 Tabs、Table、FilterDropdown 等核心组件），通过严格对齐底层设计数据，实现了对 Figma 设计稿的 100% 像素级还原（涵盖跨平台 PingFang 字体栈本地化、精确的 2px 呼吸字间距以及精准切分的列宽排版）。在视觉与交互表现上，我们使用毛玻璃兼容性与 Figma 模糊半径换算差异，使用底层取色的 `rgba(0,0,0,0.2)` 结合 `blur(12px)` 打造了实现毛玻璃效果弹窗，并通过严格的 z-index 层级上下文隔离解决了潜在的事件遮挡。

## 要求

- 允许固定外部容器宽高
- 在容器内部100%还原设计稿中的表格页
- 可以使用框架/组件库

## Quick Start (如何运行)

1. 安装依赖: `npm install`
2. 启动开发服务器: `npm run dev`
3. 浏览器访问终端输出的本地地址 (通常为 `http://localhost:5173`)
4. 字体通过 `@fontsource/noto-sans-sc` 库以本地形式离线加载。
